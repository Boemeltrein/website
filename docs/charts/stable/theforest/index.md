# theforest

![Version: 3.0.11](https://img.shields.io/badge/Version-3.0.11-informational?style=flat-square) ![Type: application](https://img.shields.io/badge/Type-application-informational?style=flat-square) ![AppVersion: theforest](https://img.shields.io/badge/AppVersion-theforest-informational?style=flat-square)

This Docker will download and install SteamCMD. It will also install The Forest and run it. SERVERNAME: 'The Forest Docker' SERVERPASSWORD: 'Chart' ADMINPASSWORD: 'adminChart' (you can change this in your SERVERFOLDER/config/config.cfg)

- https://github.com/truecharts/charts/tree/master/charts/incubator/theforest
- https://hub.docker.com/r/ich777/steamcmd
