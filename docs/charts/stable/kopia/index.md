# kopia

![Version: 8.1.4](https://img.shields.io/badge/Version-8.1.4-informational?style=flat-square) ![Type: application](https://img.shields.io/badge/Type-application-informational?style=flat-square) ![AppVersion: 0.15.0@sha256:915804f85364d2bba51248ea6b639d0988933870dc475da71d028aa819ec8649](https://img.shields.io/badge/AppVersion-0.15.0@sha256:915804f85364d2bba51248ea6b639d0988933870dc475da71d028aa819ec8649-informational?style=flat-square)

Kopia is a simple, cross-platform tool for managing encrypted backups in the cloud. It provides fast, incremental backups, secure, client-side end-to-end encryption, compression and data deduplication.
